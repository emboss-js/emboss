Core

Glass-finished bars

The signature visual. Every bar is two layers — a track at full width and a fill at progress width — with a glass highlight, dimensional shadow, and progress marker dot.

Progress marker at fill edge (hidden at 0% and 100%). Smart labels: inside when bar is wide enough, outside when narrow. Drag handles appear on hover. Done bars dim to 45%. Upcoming bars show the full track at 50% opacity.

Dependency Arrows

SVG paths connecting dependent tasks. Start-to-start routing with right-angle joins. Hidden by default — they appear when you hover a connected bar, then fade out.

Status-based styling: solid 1.5px for active, dashed for future, lighter for completed. 2.5px connection dots at endpoints. Hover any bar to see its connections.

Core

Themes

Every visual value flows through CSS custom properties. Light and dark ship with core. Swap themes by changing one class on the container. Build your own in 20 lines of CSS.

~40 custom properties prefixed with --emboss-. Surface colors, ink hierarchy, bar fills, track styles. Dark theme swaps all values. Vivid template overrides fills only.

Views & Density

Four zoom levels — day, week, month, quarter — and three density modes. Working for daily use, presentation for client meetings, dense for when you need the big picture.

Each view × density combination maps to specific pixel values: day width, row height, bar height, label size. Swap with one method call.
