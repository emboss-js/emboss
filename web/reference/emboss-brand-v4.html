How it works

The Tiptap of Gantt charts

Frappe Gantt is the engine — rock-solid date math, drag handling, dependency resolution. Emboss wraps it in a design system, extension architecture, and glass-finished visual layer.

◈

Everything is a Row

Phases, tasks, milestones, subtasks — one data model, one render pipeline. Extensions enrich rows. Renderers consume them.

◎

Glass-finished bars

Track + fill + glass highlight + shadow. Progress markers, smart labels, drag handles. Every bar looks designed by hand.

⟐

Extensions, not forks

Today markers, tooltips, arrows, sidebars, milestones — all extensions. Register what you need. Override what you want.

◆

Themes via CSS vars

Every visual value flows through custom properties. Light, dark, vivid — swap a class, change the feel. Build your own in 20 lines.
