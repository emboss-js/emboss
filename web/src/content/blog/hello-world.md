Open source builds the engine. Emboss brings the finish.

We started with a question: *why do open source tools that do incredible things under the hood still look like they were designed in 2014?*

Frappe Gantt is one of the best open source Gantt chart libraries out there. Rock-solid date math, drag handling, dependency resolution. But the visual layer hasn't kept up with what users expect from modern software.

Emboss wraps Frappe's engine in a design system, extension architecture, and glass-finished visual layer. Same power, new presence.

## What we believe

- **Beautiful software isn't a luxury.** It's accessibility. When software looks and feels right, more people can use it — and want to.
- **Elevation, not demolition.** We build on what exists. We honor the foundations.
- **The surface is where the human meets the tool.** That surface matters.

## What's coming

- Glass-finished bars with progress markers and smart labels
- Light, dark, and vivid themes via CSS custom properties
- Extension architecture for tooltips, dependency arrows, sidebars, and more
- Organize bundle for teams that need structure

This is v0.1. We're just getting started.
