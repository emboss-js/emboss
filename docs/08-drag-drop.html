Drag & Drop

Core

Two modes: move (reschedule) and resize (change duration). Handles appear on hover. Ghost shows original position.

1Hover Handles

When hovering a bar, drag handles appear on left and right edges. These allow resizing.

Default — no handles
Hovered — handles appear
Handle size6×18px, radius 3px
Handle colorrgba(255,255,255,.9)
Handle shadow0 1px 3px rgba(0,0,0,.15)
Left handleInset 4px from bar left edgeResizes start date
Right handleInset 4px from fill right edgeResizes duration
Cursorew-resize (on handles), grab (on bar body)
Appear animationopacity 0→1, 0.15s ease

2Move Mode (Reschedule)

Dragging the bar body moves the task to a new date. Ghost shows original position.

Cursorgrabbing

3Resize Mode (Change Duration)

Dragging a handle changes start date (left) or duration (right). Bar stretches in real time.

Resizing right handle — extending duration
Active handle1.5px border in bar's status color (not always blue)
Bar responseTrack width changes in real time
SnapSnaps to day boundaries
Minimum width1 day (prevents zero-width bars)
Duration tooltipShows "+2 days" or "−1 day" relative change
