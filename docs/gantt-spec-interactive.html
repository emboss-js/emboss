Architecture

Four layers, one npm package. Premium features are license-gated, not hidden.

1

Engine — Frappe v1 Mechanics

SVG rendering, date math, drag-drop. Proven mechanics, new paint job.

2

Design System — Beautiful Defaults

Glass bars, hue-shift gradients, phase grouping, tooltips, animations. The Working View ships free.

3

Theming API — CSS Custom Properties

Override --gantt-* variables for colors, spacing, radii. Extension points for renderers.

4

Premium Layer — Templates + Components

Glass, Presentation, Dense, Dark, Minimal templates. Subtasks, Critical Path, Export components.

📦 Single Package

One npm package ships everything. Premium modules produce a build-time console.warn without a valid license key. Nothing breaks.

// Free — just works import { GanttChart } from '@gantt-lib/core' // Premium — add license key import { glassTheme } from '@gantt-lib/core/themes/glass' const chart = new GanttChart({ theme: glassTheme, licenseKey: 'GANTT-XXXX-XXXX-XXXX' })

Free vs. Premium

Free = complete and beautiful. Premium = depth and variation.

Free Principles

If Frappe v1 does it → styled version is free. If required for readability → free. One gorgeous Working View.

Design variations → premium. New algorithms → premium. Hierarchy beyond one level → premium.

Roadmap

Three phases. Ship core first, then premium, then sustain.

Months 1–3

Phase 1: Core Library

Fork Frappe v1, build design system, phase grouping, sidebar, glass bars, milestones, dependencies, tooltips, drag-drop, theming API. Ship to npm. Integrate with Leantime.

Months 3–6

Phase 2: Premium & Launch

Build all templates + components. License system. Static website with live demos. Payment integration. Documentation. Product Hunt / HN launch.

Ongoing

Phase 3: Sustain

Community PRs, bug fixes. ~1 new template or component per quarter. Target $5–10K/month revenue covering part-time maintenance for Gloria + Marcel.
